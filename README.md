## General info

This project is a frontend task, to create a todo app with drag and drop for task orginizing.

## Technologies

Project is created with:

- HTML version: 5
- CSS version: 3
- JavaScript: ES 6

## Setup

To run this project, install it locally using npm:

```
$ cd ../lorem
$ npm install
$ npm start
```
